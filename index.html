Types, Strengths, and Weakness
Trainer Daniel Evans
Snorlax
Snorlax is a normal type pokemon.
Strengths: The normal type is the only type that is not super effective against any other type. Weaknesses: Snorlax is weak against fighting type pokemon.
Gengar
Gengar is a Ghost and Posion type pokemon.
Strengths: Gengar is strong against grass, fairy, ghost, and psychic type pokemon. Weaknesses: Gengar is weak against ground and dark type pokemon.
Eevee
Eevee is a normal type pokemon.
Strengths: The normal type is the only type that is not super effective against any other type. Weaknesses: Eevee is weak against fighting type pokemon.
Waillord
Wailord is a water type pokemon.
Strengths: Wailord is strong against fire, ground, and rock type pokemon. Weaknesses: Wailord is weak against grass and electric type pokemon.

The Broken Beach Story

Broken Beach Gym was founded in June 2022, just months before the catastrophic Broken Beach Earthquake set real estate development back a generation. We have decided to embrace our permanent "Under Repairs" status as part of the exciting theme of our official Pokemon Trainers League experience. Watch your step!
